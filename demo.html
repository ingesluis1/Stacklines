Live demo

Een AI-klantenservice die écht werkt

Dit is een werkende demo voor een fictieve plantenwinkel, De Groene Stuiver. De assistent zoekt in een kennisbank én pakt de juiste klantgegevens erbij. Precies zoiets bouw ik voor jouw bedrijf, op je eigen server.

Probeer bijvoorbeeld

Waar is mijn bestelling GS-1002?
Mijn Monstera krijgt gele blaadjes, wat nu?
Hoe kan ik iets retourneren?
Welke planten kunnen tegen weinig licht?

Hoe werkt dit

1
Begrijpt de vraag
Zoekt op betekenis, niet op trefwoord.
2
Pakt de data erbij
Herkent een bestelnummer en haalt de status op.
3
Schrijft een antwoord
Een AI-model formuleert een persoonlijk antwoord.
Let op: deze demo draait volledig op een eigen server (self-hosted AI, zonder cloud). Een antwoord kan daardoor 10 tot 20 seconden duren. Het lokaal gedraaide model is bovendien aanzienlijk lichter dan de grote commerciële modellen, waardoor de antwoorden hier minder verfijnd zijn dan technisch haalbaar is. In de praktijk koppel ik desgewenst een krachtiger of sneller model.
